# Python Introduction Course

A short, hands-on introduction to Python. Learn the language basics by completing and
running small notebooks, then move on to data science and machine learning.

## Notebooks

1. [Variables, Types, and Operators](notebooks/01_variables_and_types.ipynb)
2. [Control Flow](notebooks/02_control_flow.ipynb)
3. [Functions](notebooks/03_functions.ipynb)
4. [Data Structures](notebooks/04_data_structures.ipynb)
5. [Strings](notebooks/05_strings.ipynb)
6. [Files and Exceptions](notebooks/06_files_and_exceptions.ipynb)
7. [Modules and Classes](notebooks/07_modules_and_classes.ipynb)
8. [Intro to numpy and pandas](notebooks/08_intro_to_data_science.ipynb)
